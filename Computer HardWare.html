This is the Computer HardWare page

  • This is a motherboard. Motherboards allow the CPU the RAM and all other comuter hardware parts to be able to communicate with eachother.
  • This is a CPU.The CPU(Central Processing Unit) is the brains of the computer this is where most calculations are done. The CPU itself does not execute tasks, it sends out signals to the other parts of the computer to do so.
  • This is RAM. RAM(Random Access Memory) is basically a storage unit where information can be easily stored and accessed. It can be relate to the human form of memory in the way that we have short term and long term memory. Simple one time things to remember may be forgotten quickly while important things that will be used down the road will most likely be remembered for a long time. It is the same for RAM.
  • This is a Hard Drive. Computer Hard Drives are data storage devices that are usabl for storing and retrieving digital information.
  • Components on computer are related and able to communicate with one another due to thin connections named 'bus' in the motherboard
  • This is a keyboard. Keyboards are used to type and input things into the computer that the user wants. When the user interacts with the computer the keyboard is almost always used to almost everything he commands it to.
  • This is a computer mouse. A mouse allows the user to select thins on the computer. For example, click on links, closing tabs etc. There is a mouse built in on most laptops(trackpad) but there is not one built in on most desktops.
  • This is a monitior. Monitors allow you to see what you are doing on the computer. For example, they allow you to see what you are typing , where your mouse is etc. Again they come built in on laptops but most desktop computers require you to by things seprately.
